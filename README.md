# AgroConnect

AgroConnect is a MERN marketplace that connects farmers directly with consumers.

## Apps

- `client`: React + Vite + Tailwind CSS frontend
- `server`: Node.js + Express + MongoDB backend

## Quick Start

### Client

1. `cd client`
2. `npm install`
3. `npm run dev`

### Server

1. `cd server`
2. `npm install`
3. Copy `.env.example` to `.env`
4. `npm run dev`
